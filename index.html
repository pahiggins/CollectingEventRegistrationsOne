Collecting Event Registrations - Manually

Step 1

Select the ‘Events’ tab from the header.

Step 2

On the 'Events' page select an 'Event Name' to navigate to the ‘Event Dashboard’ page.

Step 3

On the ‘Event Dashboard’ page select the ‘Manage Guests’ button to navigate to the ‘Manage Guests’ page.

Step 4

On the 'Manage Guests' page select the ‘Add Guest’ button.

In the 'Add Guest' pop up add ‘First Name’, ‘Last Name’ and ‘E-mail’ and select the ‘Add’ button.

Note: the ‘E-mail’ field is optional, however, this is required if you want to send the associated guest an e-mail invitation.

Step 5

Select the ‘Register Guest’ action in the dropdown menu for the guest you have added to the event.

Step 6

This is the beginning of the event registration process. Select the ‘RSVP’ button.

Step 7

Using the green downward arrow in the bottom right-hand corner of the page navigate your way through the event registration process for the guest providing their menu selections and dietary / allergy requirements (optional) when prompted.

When you've successfully completed the event registration process you'll be presented with this page.

Step 8

Close the current browser tab to return to the ‘Manage Guests’ page where you'll find the registration details for the respective guest have automatically updated.
